Project Description

This project implements the popular puzzle game 2048 on the TI CC3200 embedded microcontroller. The game runs fully on-device, rendering the board on an Adafruit SSD1351 OLED display and allowing input via either UART or accelerometer-based tilt control.

Core features include:

  • Real-time tile movement and merging
  • Random tile spawning
  • OLED rendering with dynamic number formatting
  • Accelerometer tilt input handling with debounce logic
  • Post-game score submission to AWS IoT via HTTPS

Planned features like temperature-based tile coloring were scoped but not implemented due to hardware integration constraints.
